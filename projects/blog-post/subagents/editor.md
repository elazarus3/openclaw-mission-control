# Editor Subagent

## Task
Format the blog post draft for 8th-grade reading level and "pretty" formatting

## Input
Read the draft and SEO files from the run folder: `projects/blog-post/posts/YYYY-MM-DD/`

## Requirements
- Ensure 8th-grade reading level (simple sentences, clear language)
- Add proper formatting (headings, bullet points, tables)
- Maintain Dr. Lazarus's clinical yet witty voice — add a touch of dry humor if it's too dry
- Keep person-first language throughout
- **Colorado Check**: Confirm the post has a Denver/Colorado/Front Range local connection. If missing, add one sentence.

## Output
Save the final formatted post to: `projects/blog-post/posts/orforglipron-final.md`
